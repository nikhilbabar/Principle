Introduction

MatrixSearch engine provides powerful searching against any string, which makes the precise way of searching nearest strings, this recommendation engine brings together the power of search and spell checking, engine searches nearest word match in any unstructured string.

MatrixSearch corrects even spelling mistakes with unmatched accuracy. spelling checker uses a Levenshtein Distance Algorithm which is based on matrix technique to correct any type of mistake and set priorities as per matching strings.

Application

  1. Provide auto string mapping against source string.
  2. Provide matching recommendation against source string.

Business Needs

Search engine is what increases your chances of being found when customers search for a product or service with Google, Bing, and other search engines.

For any custom ETL tool focus is to get unmapped entities from staging area and show best matching strings as per previous saved translations, we are writing custom ETL service to handled complex business logic with multiple file template and languages as per data provider companies.

To achieve complex file processing, we developed our own custom recommendation engine using Levenshtein Distance Algorithm.

Note

MatrixSearch engine works against text only (Currently working with only text not supporting any media files for e.g. picture, html and video).

Transform Techniques

Some of the most powerful approaches to semantic processing are based on the use of mathematical transform techniques. Matrix driven techniques have been the most successful.

We use Levenshtein Distance Algorithm which is based on matrix technique that help us produce high precision results.

MatrixSearch Processing

Compare, Refine, Evaluate and Iterate

Text Process comprises following steps

  • - Receives source text key
  • - Target text acquisition
  • - Compare, Refine, Evaluation and Iterate
    • Compare:
      • Compare source key text with target text
    • Refine:
      • Define content strategy
      • Sort as per nearest match rank (measures the effectiveness)
    • Evaluation:
      • Define content size for result set
      • Content creation
  • - Pass information for business use
